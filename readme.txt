*Cách cài đặt:
- Cài đặt streamlit:
 + Mở cmd của desktop -> pip install streamlit
 (nếu chưa có pip thì cài pip)
- Cài đặt các thư viện trong requirement.txt.
Có thể sẽ thiếu một vài thư viện thì tùy vào lỗi để cài thêm.
pip install requirement.txt
*Cách chạy chương trình:
- Vào terminal của thư mục chứa project gõ lệnh: streamlit run spamDetector.py
